# UI reference

Optional demo shell. Does not block core milestones.

## Inspiration

[Dispatch](https://dispatch.framer.website/) — industrial / editorial product UI: dark charcoal frame, cream content panels, thin grid borders, hatched dividers, asymmetrical clipped corners, numbered sidebar nav.

## Zyocra accent rule

**Replace all orange accents with a refined yellow** (e.g. `#F5C518` or `#FFD000`).

Yellow is reserved for:

- Logo blocks
- Primary CTAs
- Active indicators
- Chart highlights tied to the accent

Do not invent a second accent palette.

## Preserve from the reference

- Dark near-black page background
- Large cream / off-white panels (`#E8E3D5`–`#F2F0EB`)
- Thin borders, diagonal hatch patterns, subtle dot grids
- Chamfered / clipped corners on logo and key CTAs
- Bold headings, clean sans UI text, monospace labels
- Generous whitespace and modular cards

## Suggested surfaces (when built)

- Epoch risk score and bucket
- Proof status (baseline vs custom path)
- Oracle submission metadata
- Consumer collateral parameters after update
- Benchmark headline table

Stack preference when scaffolded: **Vite + React**, local `pnpm dev` only.
