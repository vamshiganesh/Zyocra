Zyocra is a benchmark-driven zkML oracle for DeFi. It compares compiler-generated EZKL circuits against a hand-optimized Circom LoRA head on the same hidden-to-logit statement, verifies scores on EVM, and updates mock lending policy by risk bucket. Local-first. No paid infra required for the default path.
